## AQI Simple Reflex Agent Architecture

```
Environment
    │
    ▼
Sensors (Pollution Data File)
    │
    ▼
Agent
    │
    ▼
Rules (AQI Calculation)
    │
    ▼
Actuator
    │
    ▼
Display AQI Category
```

The agent reads environmental parameters from a dataset (sensor input file), 
applies AQI calculation rules, and outputs the AQI category for the region.
